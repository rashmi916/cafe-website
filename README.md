# ☕ Café Website

A responsive and visually appealing café website built using HTML, CSS, and JavaScript. This project showcases a modern UI design for a café, including menu display, contact section, and smooth user experience.

---

## 🚀 Features

- 🏠 Home page with attractive banner
- 📖 Menu section with food & beverages
- 📍 About Us section describing the café
- 📞 Contact section with form
- 📱 Fully responsive design (mobile, tablet, desktop)
- ✨ Smooth scrolling and basic interactivity using JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
